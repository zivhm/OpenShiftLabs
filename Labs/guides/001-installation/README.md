# 001-Installation — Core OpenShift/OKD installation and CLI tools

This module focuses on installing OpenShift/OKD components, CLI tools, and essential tooling needed for labs.

## Objectives

- Learn installation options for OpenShift / OKD and where to get official images.
- Install required CLI tooling (oc, kubectl, helm, yq) and container builder tools (Buildah/Podman/ Docker).
- Understand basic cluster configuration and testing steps.

## Tasks

1. Follow official installation docs to bootstrap a local or cloud OpenShift/OKD cluster.
2. Install CLI tools: `oc`, `kubectl`, and `helm`.
3. Verify an admin user and access to the console.
4. Test: run sample `oc` commands such as `oc get nodes`, `oc get projects`.

Estimated time: 30–60 minutes
