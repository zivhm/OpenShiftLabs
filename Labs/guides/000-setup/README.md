# 000-Setup — Accessing OpenShift web console

This module covers accessing the OpenShift web console and verifying cluster connectivity.

## Objectives

- Access the OpenShift web console.
- Navigate the console and verify cluster access.
- Create your first project.

## Tasks

1. Access the OpenShift web console and log in.
2. Verify cluster access by viewing nodes in the web console (Administrator view → Compute → Nodes).
3. Create a practice project using the web console (Developer view → Project dropdown → Create Project).

Estimated time: 30–45 minutes
