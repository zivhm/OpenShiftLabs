# How-to Guides

Practical guides for common tasks and scenarios.

## Available Guides

Modules available in this folder (ordered learning path):

- `000-setup` — Accessing OpenShift web console
- `001-new-user` — Creating and managing users and roles
- `002-new-project` — Creating namespaces/projects with resource quotas and limits
- `003-docker-lifecycle` — Building, tagging, and managing local Docker images
- `004-docker-pipeline` — CI pipeline basics; building from source to image
- `005-hooks-setup` — Git/build hooks and webhooks for CI triggers
- `006-images-imagestream` — BuildConfigs and ImageStreams, registry interactions
- `007-deploying` — Deployment vs DeploymentConfig, scaling and rollouts
- `008-services-routes` — Exposing services using Routes and Ingress
- `009-monitoring` — Observability with Prometheus and Grafana
