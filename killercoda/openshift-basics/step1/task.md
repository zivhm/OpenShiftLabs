# Setup and Open Labs

1) Initialize the environment (auto-run via curl | bash):
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/hitibash/OpenShiftLabs/feature/killercoda-generator/killercoda/openshift-basics/assets/setup.sh)"
```

Alternative (source into current shell):
```bash
source <(curl -fsSL https://raw.githubusercontent.com/hitibash/OpenShiftLabs/feature/killercoda-generator/killercoda/openshift-basics/assets/setup.sh)
```

2) Open the labs site:
- https://hitibash.github.io/OpenShiftLabs/

Follow the steps on the site. Use this terminal to run commands as instructed.
