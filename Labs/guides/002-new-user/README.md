# 002-New User — Creating and managing users and roles

This module teaches how to create and manage developer and operator users, project roles, and RBAC policies in OpenShift.

## Objectives

- Create users or service accounts for developers and operators.
- Assign roles and permissions using RBAC (RoleBindings and ClusterRoleBindings).
- Understand best practices for least-privilege access and multi-team collaboration.

## Tasks

1. Create a developer and operator user.
2. Assign roles using `oc adm policy add-role-to-user` and check access with `oc auth can-i`.
3. Demonstrate the difference between RoleBindings and ClusterRoleBindings.

Estimated time: 20–40 minutes
